--- layout: publication year: 2025 title: "Embedding Atlas: Low-Friction, Interactive Embedding Visualization" authors: - Donghao Ren - Fred Hohman - Halden Lin - Dominik Moritz venue: VIS venue_location: Vienna, Austria venue_url: https://ieeevis.org/year/2025/welcome venue_tags: - VIS type: - Conference - Short Paper tags: - Visualization - Machine Learning - Embeddings link: https://apple.github.io/embedding-atlas/ arxiv: "2505.06386" --- Embedding visualizations are essential for understanding high-dimensional data in machine learning, yet existing tools often present significant usability challenges. We introduce Embedding Atlas, an interactive platform designed to minimize friction in embedding visualization workflows. The system addresses key pain points including tedious data wrangling and loading, scalability limits, and lack of workflow integration. Embedding Atlas leverages modern web technologies alongside algorithmic approaches like density-based clustering and automated labeling to enable efficient analysis of large-scale embeddings. Through performance benchmarks and competitive comparisons with existing tools, we demonstrate how our open-source system provides a more accessible and scalable solution for exploring and understanding embedding spaces in machine learning workflows.
